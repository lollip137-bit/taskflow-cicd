Help Guide

Manage your daily tasks easily with TaskFlow.

TaskFlow is a lightweight task-management web application created for DevOps Fundamentals Assignment 04.

Use Dashboard to view summary, Add Task to create a task, Task List to view tasks, and Task Detail to view selected task information.

This project demonstrates GitHub branching, pull requests, Docker image build, CI workflow, CD workflow, and deployment using Render.
